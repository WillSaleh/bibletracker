# Bible Tracker

A full stack web app for tracking your daily Bible readings, building streaks, and visualizing your consistency over time

## Tech Stack

**Frontend**
- Next.js (App router)
- React
- Typescript
- Tailwind CSS

**Backend**
- Python
- FastAPI
- SQLModel
- SQLite

**Auth**
- JWT Tokens
- Password hashing with passlib

## Features

- Register and log in with email + password
- Log Bible passages you've read each day 
- View your current reading streak
- Github style heatmap of your reading history 
- Full reading history with delete support
